# TixSigurado (Hackathon MVP)

## Overview
A Stellar + Soroban-based ticketing system that prevents fake tickets and scalping.

## Blockchain
- Stellar Testnet only
- Soroban smart contracts

## Core Features
- Freighter wallet connection
- Ticket minting (testnet)
- Ticket validation
- Resale with restrictions

## Key Demo Flow
1. Connect wallet
2. Mint ticket
3. View ticket
4. Validate ticket (QR simulation)

## What is implemented
- Wallet connection
- UI from Stitch MCP
- Basic contract interaction layer

## What is NOT implemented
- Mainnet deployment
- Full backend
- Real event integrations

## Goal
Working hackathon demo only
